Tecnologias de Programação Web

Back-end

  • Laravel - Ano de criação: 2011 - Criador: Taylor Otwell - laravel.com - Framework PHP elegante e expressivo.
  • ASP.NET - Ano de criação: 2002 - Criador: Microsoft - dotnet.microsoft.com - Framework web open source para construir aplicativos modernos na plataforma .NET.
  • Spring MVC - Ano de criação: 2002 - Criador: Pivotal Software - spring.io - Framework Java para construção de aplicativos web escaláveis e de alta qualidade.
  • Express - Ano de criação: 2010 - Criador: TJ Holowaychuk - expressjs.com - Framework Node.js para construção de aplicativos web robustos e escaláveis.
  • Django - Ano de criação: 2005 - Criador: Django Software Foundation - djangoproject.com - Framework Python para construção de aplicativos web, que incentiva o desenvolvimento rápido e limpo.

Front-end

  • Angular - Ano de criação: 2010 - Criador: Google - angular.io - Framework JavaScript para construção de aplicativos web escaláveis e de alta performance.
  • React - Ano de criação: 2013 - Criador: Facebook - reactjs.org - Biblioteca JavaScript para construção de interfaces de usuário declarativas e escaláveis.
  • Vue - Ano de criação: 2014 - Criador: Evan You - vuejs.org - Framework JavaScript progressivo para construção de interfaces de usuário interativas e dinâmicas.
  • Svelte - Ano de criação: 2016 - Criador: Rich Harris - svelte.dev - Framework JavaScript radicalmente novo para construção de aplicativos web rápidos e eficientes.

Outras

  • WordPress - Ano de criação: 2003 Criador: Matt Mullenweg e Mike Little - wordpress.org - Plataforma de gerenciamento de conteúdo (CMS) de código aberto e gratuito, utilizada para criar sites e blogs.
  • OutSystems - Ano de criação: 2001 - Criador: Paulo Rosado - outsystems.com - Plataforma de desenvolvimento de aplicativos de baixo código para construir aplicativos web, mobile e empresariais.
  • Weebly - Ano de criação: 2006 - Criador: David Rusenko, Chris Fanini e Dan Veltri - weebly.com - Plataforma de construção de sites e comércio eletrônico com interface simples e intuitiva.
  • Wix - Ano de criação: 2006 - Criador: Avishai Abrahami, Nadav Abrahami e Giora Kaplan - wix.com - Plataforma de construção de sites com ferramentas de design visual para criar sites web e lojas online.

Tendências modernas de programação Web

Algumas das tendências mais populares incluem:

  • Desenvolvimento orientado a componentes
  • Integração de inteligência artificial e aprendizado de máquina
  • Progressive Web Apps
  • Aplicações de realidade virtual e aumentada

Aspectos obsoletos da programação Web

Alguns aspectos obsoletos incluem:

  • Uso excessivo de imagens em vez de CSS para efeitos visuais
  • Layouts baseados em tabelas em vez de CSS flexbox ou grid
  • Desenvolvimento sem consideração de acessibilidade
